Web Development - Semester 1

In this module, you will learn about the front-end of Web Development, in particular, how to use a scripting language such as JavaScript to control the web page and make it more dynamic. Please begin by checking out the Module Guide section, and also have a look at the assessment schedule. Of course, if you have any questions to ask, you will find a list of people you could contact, in the Module Guide section.

Week 1 Learning Objectives

  • Explain the concepts of the Internet, the web, frontend, backend, and database components in the context of web development.
  • Differentiate between frontend developers, backend developers, and full-stack developers, and identify the scripting and programming skills necessary for each role.
  • Identify various frameworks used in web development and discuss their advantages.
  • Understand the importance of mastering fundamental frontend development tools like HTML, CSS, and JavaScript to become an effective and efficient web developer.
  • Describe the role of HTML in web development and write basic HTML scripts for demonstration purposes.
  • Explain the function of CSS in web development and create simple CSS scripts for demonstration.
  • Articulate the significance of JavaScript in web development.
